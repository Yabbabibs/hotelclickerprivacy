Request Data Deletion

If you would like to request deletion of any data associated with your use of the Hotel Clicker app, please follow the instructions below:

How to Request Deletion:

  1. Send an email to: srhtnkykz@gmail.com
  2. Use the subject line: Data Deletion Request - Hotel Clicker
  3. Include any identifying information that can help us locate your data (such as advertising ID, if available).

Data We Can Delete:

  • Game progress or identifiers stored on our servers (if any)
  • Advertising identifiers (if collected)

Data Retention Policy:

We do not store personal user data beyond what is necessary for game functionality. Data that is not tied to an identifier or account is automatically discarded.

If you have further questions, feel free to contact us at srhtnkykz@gmail.com
